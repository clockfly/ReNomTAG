# FR.02 Data Flow Diagram(DFD)


#### 1. クリックイベントからボックス作成まで

```mermaid
  graph TD
    subgraph image_canvas.vue
      currentBbox
      onMouseDown
      onMouseMove
      onMouseUp
      onMouseDown-->appendBbox
      onMouseDown--create_new_obj-->currentBbox
      onMouseUp--null-->currentBbox
      onMouseUp-->updateBbox
    end

    subgraph box.vue
      onMouseMove-->createdScalingBox
      onMouseMove--true-->selectedFlag
      subgraph box_props
        x
        y
        w
        h
        selectedFlag
      end
      createdScalingBox-->x
      createdScalingBox-->y
      createdScalingBox-->w
      createdScalingBox-->h
    end
  end
``` 

#### 2. ボックス選択

```mermaid
  graph TD
    subgraph image_canvas.vue
      currentBbox 
      onMouseDown
      onMouseMove
      onMouseUp
      onMouseUp-->id{If currentBbox}
      id--yes-->appe
    end

    subgraph box.vue
      onMouseMove-->moveBox
      subgraph box_props
        x
        y
        w
        h
        selectedFlag
      end
    end

    onMouseDown--select_one_obj-->currentBbox
    onMouseDown--true-->selectedFlag
 
  end
```
