# Error Codes

| コード | メッセージ      | 概要                                     |
|--------|-----------------|------------------------------------------|
| 0      | -               | 成功                                     |
| 1      | File not found. | 指定したフォルダやファイルが見つからない |
| 2      | Time out.       | サーバからの返答待ちタイムアウト         |
| 999    |                 | 未知のエラー                             |
